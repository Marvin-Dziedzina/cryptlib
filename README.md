# CryptLib
CryptLib is a small and simple to use cryptography crate for Rust that uses OpenSSL.

## Capabilities
- RSA encryption
- AES encryption
- SHA 256, 384, 512

### Getting Started
Include this section into your Cargo.toml.
```Cargo.toml
[dependencies.cryptlib]
git = "https://github.com/Marvin-Dziedzina/cryptlib"
```
